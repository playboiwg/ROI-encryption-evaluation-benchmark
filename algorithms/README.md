# Baseline Algorithms

This folder contains refactored, ready-to-run implementations of representative ROI encryption baselines.

| Method       | Status                  | Download |
|--------------|-------------------------|----------------------------|
| Yu et al.    | Available            | [Yu_et_al_code.zip](https://github.com/playboiwg/ROI-encryption-evaluation-benchmark/releases/download/v1.0.0/Yu_et_al_code.zip) (93 MB) |
| Taha et al.   | Available            | [Taha_et_al_code.zip](https://github.com/playboiwg/ROI-encryption-evaluation-benchmark/releases/download/v1.0.0/Taha_et_al_code.zip) (101 MB) |

Both implementations are based on HM-16.9 and support standard-compliant bitstream output.
